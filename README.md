This is a restaurant review app that uses ruby and react to allow users to create, read, update and delete reviews on various restaurants.


The backend server runs on : http://localhost:9292/reviews

Use 'npm install' and 'npm start' commands to get the react app running.
The react app runs on : http://localhost:3000/?



Link to the frontend repository is: https://github.com/MelynAtieno/sinatra-review-frontend
